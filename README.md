# 🧠 Abalone Age Prediction using Machine Learning

This project predicts the age of abalone shells using physical measurements
such as length, diameter, and shell weight.

## 🚀 Live Demo
https://abalone-predictor.onrender.com

## 🛠 Tech Stack
- Python
- Flask
- Scikit-learn
- HTML / CSS / Bootstrap

## 📊 Machine Learning Pipeline
1. Data preprocessing
2. Feature encoding
3. Model training
4. Model evaluation
5. Deployment using Flask

## 🔍 Features
- Interactive web UI
- REST API endpoint
- Real-time prediction
- Cloud deployment
